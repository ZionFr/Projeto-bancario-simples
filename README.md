Projeto Bancário em Java

Projeto simples em Java criado para fins de estudo. Usei conceitos fundamentais como classes, objetos, métodos e encapsulamento.

Funcionalidades:

• Criação de contas bancárias
• Depósito de valores
• Saque de valores
• Exibição de dados da conta

Tecnologias utilizadas:

• Java 22

Estrutura do projeto:

src/
  projetobancario/
    Conta.java
    Main.java

Como executar:

Abra o terminal na pasta raiz.

Compile os arquivos Java:
javac src/projetobancario/*.java

Execute o programa:
java src.projetobancario.Main
